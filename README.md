# **React VAC Assignment 2**

User Data Collection using **React.useState()**

## **To Run this Project**

```bash
git clone https://github.com/git4robin/ReactAssignment1.git

# Step1
cd ReactAssignment1
# Execute this command to download all node modules
yarn
# OR
npm install

# Step2
# Launch the development server
yarn start
# OR
npm start
```
